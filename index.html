Intro

This website demonstrates the attack chain experiments from the paper Parasites in the Toolchain: A Large-Scale Analysis of Attacks on the MCP Ecosystem. All specific MCP server information and author information have been anonymized for privacy and security purposes.

Note: All test accounts used in this study belong to our own team, and no real-world systems were targeted or compromised.

Experiment of Chain1

Within Cursor, the user simultaneously connected to Server 1 with External Ingestion Tool 1, Server 2 with Privacy Access Tool 2, and Server 3 with Network Access Tool 3.

Attack Flow:

  1. The user sends a normal prompt that invokes Tool 1.
  2. Tool 1 inadvertently retrieves an external resource containing malicious prompts.
  3. The malicious prompts in the context cause Cursor to perform unauthorized actions.
  4. Tool 2 is invoked to read the local MCP configuration file.
  5. Tool 3 sends the configuration information via email to the attacker.
